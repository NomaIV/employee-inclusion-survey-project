# Analysis Workbook Sample

## Overview
This Excel workbook demonstrates the data analysis framework used in the Employee Wellness Survey project.

## Key Features Demonstrated:
- **Survey Results Dashboard** – Interactive visualization of response data
- **Departmental Analysis** – PivotTables comparing sentiment across teams  
- **Trend Identification** – Time-based analysis of participation patterns
- **Automated Reporting** – Dynamic formulas for real-time insights

## Sample Data Preview:
*Data structure includes anonymized employee responses across 5 departments with metrics on:*
- Overall satisfaction (1-5 scale)
- Work environment feedback
- Wellbeing indicators
- Open-text sentiment analysis

## Formulas Highlighted:
- `=AVERAGEIFS()` – Department-specific sentiment scores
- `=COUNTIFS()` – Participation rates by team
- `=XLOOKUP()` – Employee data validation
- PivotTables – Multi-dimensional analysis
